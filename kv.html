Осталась ли старая версия файла «index.html» с приветствием на английском языке в главной ветке «main»?

Да, старая версия файла «index.html» с приветствием на английском языке осталась в главной ветке «main».

В какой ветке после выполнения задания вы физически находитесь?

После выполнения задания я физически нахожусь в ветке dopZadaniye.

С помощью какой команды переключаются ветки в Git?

Для переключения между ветками используется команда:

git checkout имя_ветки

или более новый способ:

git switch имя_ветки
